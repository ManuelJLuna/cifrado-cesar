# Cifrado César

### ¿En qué consiste?

El cifradi César, nombrado así por el emperador romano Julio César, consiste de rotar las letras una cierta cantidad de veces en el abecedario.
Por ejemplo, si tenemos una "A" y tenemos una rotación de 1, nos quedará una "B".
Si la rotación es de 2, tendremos una "C". Así consecutivamente.
Al llegar a la "Z" volveremos hasta la "A".

### ¿Cómo usarlo?

En el primer input se debe colocar el texto original, en el segundo saldrá el texto cifrado

Simplemente hace falta clonar el repositorio y abrir el archivo *index.html*
